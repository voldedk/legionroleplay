HVAD ER LEGION ROLEPLAY?

En dansk FiveM roleplay server bygget på fællesskab, kvalitet og sjov

FÆLLESSKAB

Bliv en del af vores venlige og inkluderende community. Her er plads til alle, uanset erfaring med roleplay.

DANSK TALE

Vi bruger dansk tale på serveren for den mest autentiske og fordybende roleplay oplevelse.

UNIKKE FEATURES

Oplev custom scripts, jobs, køretøjer og meget mere der gør vores server speciel og spændende.

AKTIV STAFF

Vores dedikerede staff team sikrer en fair og sjov oplevelse for alle spillere.

CUSTOM KØRETØJER

Kør rundt i unikke og realistiske køretøjer som du ikke finder på andre servere.

MANGE JOBS

Fra politi til mekaniker - vælg din egen karrierevej og skab din egen historie.

JOIN VORES DISCORD

Bliv en del af vores voksende community! På Discord kan du møde andre spillere, holde dig opdateret med nyheder, og få hjælp fra vores staff team.
